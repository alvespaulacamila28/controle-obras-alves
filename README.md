# Controle de Obras Alves

Sistema simples para controle de obras, orçamentos, gastos e anotações.

## O que já funciona

- Cadastro de clientes
- Cadastro de obras
- Lançamento de orçamento por material/item
- Controle de gastos por obra
- Anotações diárias da obra
- Resumo geral com clientes, obras, gastos e lucro/prejuízo
- Layout simples pensado para celular

## Como abrir

Este primeiro modelo está em um único arquivo chamado `index.html`.

Para testar:

1. Abra o arquivo `index.html` no navegador.
2. Cadastre um cliente.
3. Cadastre uma obra para esse cliente.
4. Lance orçamento, gastos e anotações.

Os dados ficam salvos no navegador do aparelho usando armazenamento local.

## Observação

Esta é uma primeira versão simples para validar a ideia. Depois ela pode evoluir para um aplicativo com login, banco de dados online e acesso por mais de um celular.
